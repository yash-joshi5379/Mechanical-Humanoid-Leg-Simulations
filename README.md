# Mechanical-Humanoid-Leg-Simulations

This repository contains a modular MATLAB simulation suite for a multi-jointed robotic leg. The project focuses on the kinematic modeling, 3D visualization, and design optimization of three primary joint assemblies: a 3-DOF Spherical Parallel Manipulator (Hip), a Crossed Four-Bar Linkage (Knee), and a Worm-Driven Linkage (Ankle).
